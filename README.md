# 🥗 轻食热量计算器

一个基于 React 的轻食搭配与热量计算应用，帮助你科学搭配健康饮食。

## ✨ 功能特点

- 🍱 **丰富的食材数据库** - 包含 20+ 种常见轻食食材
- 🔢 **精准热量计算** - 实时计算热量、蛋白质、碳水和脂肪
- 📊 **营养成分展示** - 清晰展示各项营养指标
- 💡 **健康建议** - 根据总热量提供饮食建议
- 🎨 **现代化设计** - 紫色渐变主题，美观易用
- 📱 **响应式布局** - 完美支持移动端和桌面端

## 🚀 快速开始

### 本地运行

直接在浏览器中打开 `index.html` 文件即可使用。

### 部署到 Vercel

#### 方法一：通过 Vercel CLI 部署

1. 安装 Vercel CLI（如果尚未安装）：
```bash
npm install -g vercel
```

2. 在项目目录下运行：
```bash
vercel
```

3. 首次使用需要登录 Vercel 账户，按照提示完成配置

4. 部署完成后，Vercel 会提供一个在线访问链接

#### 方法二：通过 Vercel 网站部署

1. 访问 [Vercel](https://vercel.com)
2. 使用 GitHub/GitLab/Bitbucket 账户登录
3. 点击 "New Project"
4. 导入你的 Git 仓库
5. Vercel 会自动检测配置并部署

#### 方法三：直接拖拽部署

1. 访问 [Vercel](https://vercel.com)
2. 登录后点击 "Add New..." → "Project"
3. 选择 "Browse" 或直接拖拽项目文件夹到页面
4. 点击 "Deploy" 完成部署

## 📝 使用说明

1. **选择分类** - 点击顶部的分类标签（主食、蛋白质、蔬菜、水果、其他）
2. **添加食材** - 点击食材右侧的 "＋" 按钮添加到搭配列表
3. **调整份量** - 在已选食材中修改克数，热量会实时更新
4. **删除食材** - 点击食材右侧的 "×" 按钮移除
5. **查看结果** - 右侧会显示总热量和各项营养成分

## 🍎 食材分类

- **主食**：糙米饭、全麦面包、燕麦、紫薯、玉米
- **蛋白质**：鸡胸肉、三文鱼、鸡蛋、豆腐、虾仁
- **蔬菜**：西兰花、生菜、番茄、黄瓜、胡萝卜、菠菜
- **水果**：苹果、香蕉、蓝莓、草莓、奇异果
- **其他**：牛油果、坚果、橄榄油、酸奶

## 🎯 健康建议标准

- **< 300 kcal** - 热量偏低，建议增加主食或蛋白质
- **300-600 kcal** - 热量适中，适合作为一餐
- **600-800 kcal** - 热量较高，注意控制
- **> 800 kcal** - 热量过高，建议减少份量
